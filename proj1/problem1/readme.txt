This is the readme.txt for problem1 of project one. Here are the instructions
for compilation and execution of the program. 
 
Line command for compilation:
	$ javac pc_static.java
		or
	$ javac pc_dynamic.java

Line command for execution:
 - default number of threads (4)
	$ java pc_static
		or
	$ java pc_dynamic
 - with 14 threads
	$ java pc_static 14
		or
	$ java pc_dynamic 14
