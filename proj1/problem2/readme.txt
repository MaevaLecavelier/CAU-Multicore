This is the readme.txt for problem2 of project one. Here are the instructions
for compilation and execution of the program. You can try with mat1000.txt or mat5.txt.
 
Line command for compilation:
	$ javac MatmultD.java

Line command for execution:
 - default number of threads (6) with 500*500 matrix 
	$ java MatmultD < mat500.txt 
 - with 10 threads and 500*500 matrix
	$ java MatmultD 10 < mat500.txt
